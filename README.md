# Landing Page - אינסטלציה ודודי שמש

דף נחיתה מקצועי לעסק אינסטלציה והתקנת דודי שמש, בעברית עם תמיכה מלאה ב-RTL.

## 🚀 טכנולוגיות

- **React.js** - ספריית UI
- **TypeScript** - שפת תכנות מוקלדת
- **SCSS** - עיצוב מתקדם
- **Vite** - כלי build מהיר

## 📦 התקנה

```bash
# התקן תלויות
npm install

# הרץ בסביבת פיתוח
npm run dev

# בנה לפרודקשן
npm run build

# צפה בגרסת פרודקשן
npm run preview
```

## 🎨 עיצוב

- **צבע ראשי**: #008BFF
- **צבע משני**: #5A9CB5
- תמיכה מלאה ב-RTL (ימין לשמאל)
- Responsive לכל המסכים

## 📱 רכיבים

### Hero
דף פתיחה עם כותרת ראשית וכפתורי קריאה לפעולה

### Services
רשימת השירותים שמוצעים

### WhyChooseMe
יתרונות ונקודות מכירה

### Gallery
גלריית תמונות עבודות

### Testimonials
המלצות לקוחות

### Contact
טופס יצירת קשר עם ולידציה

### Footer
פוטר עם קישורים ופרטי התקשרות

### FloatingButtons
כפתורים צפים של WhatsApp וטלפון
